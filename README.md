# Семейное древо (Family Tree)

Веб-приложение для создания и визуализации генеалогического дерева с возможностью перетаскивания карточек, экспорта и автосохранением на сервере.

## Возможности

- **Визуализация дерева** — автоматическое размещение карточек по поколениям с линиями связей
- **Drag-and-drop** — карточки можно перетаскивать, позиции сохраняются
- **CRUD** — добавление, редактирование, удаление родственников
- **Несколько деревьев** — можно вести несколько семей одновременно
- **Экспорт/импорт JSON** — перенос данных между устройствами
- **Экспорт SVG** — сохранение дерева как картинки (экран / печать)
- **Поиск** — по имени, городу, роду
- **Цветовая маркировка** — женские и мужские карточки разных оттенков
- **Серверное хранение** — данные сохраняются на сервере, доступны из любого браузера
- **Загрузка медиа** — фото и видео для каждого члена семьи

## Стек

- **Frontend:** Vanilla JS (ES6), HTML5, CSS3 (glassmorphism)
- **Backend:** Python 3.13 + HTTP-сервер (без сторонних библиотек)
- **Хранилище:** JSON-файл на сервере + localStorage (резерв)
- **Шрифты:** Manrope, Cormorant Garamond (Google Fonts)

## Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/riddik0331/My-Family-Tree-OC.git
cd My-Family-Tree-OC

# Запустить сервер
python server.py
```

Открыть в браузере: `http://127.0.0.1:8000/`

Или запустить двойным кликом `run-server.bat` (Windows).

## Структура проекта

```
My-Family-Tree-OC/
├── app.js             # Клиентская логика (CRUD, рендеринг, drag-and-drop)
├── index.html         # Главная страница (UI)
├── styles.css         # Стили (glassmorphism)
├── server.py          # HTTP-сервер (статик + API + загрузка медиа)
├── run-server.bat     # Скрипт запуска для Windows
├── SPEC.md            # Техническое задание
├── data/              # Данные деревьев (игнорируется git)
└── media/             # Загруженные фото/видео (игнорируется git)
```

## API endpoints

| Метод | Путь | Описание |
|-------|------|---------|
| `GET` | `/api/data` | Получить все деревья |
| `POST` | `/api/data` | Сохранить деревья |
| `POST` | `/api/upload` | Загрузить медиа-файл |
